# Convertisseur

## Page HTML 

- un input de texte
- un bouton pour convertir les °C en °F
- un bouton pour convertir les °F en °C
- un `<p>` dans lequel on affichera la conversion

## JS

On met un gestionnaire d'événement sur chacun des boutons de façon à ce que la conversion se fasse quand on clique dessus.

- on récupère ce que l'utilisateur a saisi
- on fait le calcul de conversion
- on affiche le résultat dans le `<p>`.

` Exemple : 10°C équivaut à 37.56°F.`
